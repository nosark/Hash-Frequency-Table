Description: this ADT allows you to create a Hash Frequency Table. Within that table you can add items to the ADT and also keep track of the number of times the item had been entered into the table(frequency) using a hash function and quadratic probing.

Methods implemented: 

	<Object>.click();
	<Object>.count();
